---
title: Filters in Vue.js
tags: [vue.js]
---

## JavaScript

```js
// Global
Vue.filter('filterName', function(value) {
  return;
});

// Locally
new Vue({
  filters: {
    filterName(value) {
      return;
    }
  }
});
```

## HTML

```html
{{ data | filter }}
```

You can using multiple filters in Vue.js and pass additional arguments.

```html
{{ data | filterFirst | filterSecond }}
```

```html
{{ data | filterName(arg1, arge2) }}
```
