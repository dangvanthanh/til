---
title: Multiple Panel in Tmux
tags: [tmux, vim]
---

In `tmux.conf` file, add the following lines

```shell
unbind C-b
set -g prefix C-a
bind-key C-a send-prefix
bind | split-window -h
bind - split-window -v
```

## Split horizontally

```shell
ctrl+a _
````

## Split vertically

```shell
ctrl+a |
````

## Kill the current pane

```shell
ctrl+a X
```
